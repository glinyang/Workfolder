Installation Instructions:
-------------------------

Carry out the following steps on all Desktop application workstations:

1. Close all Data Collection applications (such as Author and Professional).

2. Backup the following file by adding the ".pre601p1IF" extension suffix to the file name.

   For example, the file""should be renamed ".pre601p1IF".

   

3. Run the "IBM SPSS Data Collection Desktop6.0.1 Fix Pack 1 Interim Fix (x64).exe" installer

Uninstallation Instructions:
-------------------------

Carry out the following steps on all Desktop application workstations:

1. Close all Data Collection applications (such as Author and Professional).

2. Uninstall "IBM SPSS Data Collection  Desktop 6.0.1 Fix Pack 1 Interim Fix (x64)" from the Windows Control Panel's "Add or Remove Programs" feature.

3. Restore the backup file by removing ".pre601p1IF"from the end of the file name. If the original file exists, you must first delete it to ensure that renaming will work.

   For example, the file".pre601p1IF"should be renamed "".
